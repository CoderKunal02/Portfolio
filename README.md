# Kunal Choudhary | Professional Portfolio

A sleek, modern, and interactive portfolio website designed to showcase my experience, skills, projects, and achievements as a Software Engineer and IT Specialist.

## 🚀 Features

- **Modern Glassmorphic UI**: Beautiful dark mode layout featuring dynamic glassmorphic panels and gradient borders.
- **Interactive Projects Section**: Browse filtered categories of projects, including Web Applications, Desktop Apps, and DevOps implementations.
- **Skill Visualizer**: Interactive indicator bars showing levels of expertise across core languages, backend architecture, databases, DevOps, and AI ecosystems.
- **Professional Education & Certification Timelines**: Interactive lists displaying educational background and verifyable credentials.
- **Responsive Layout**: Seamlessly adapted for all screen sizes from mobile devices to high-resolution desktop monitors.
- **Contact Forms & Direct CTA**: Integrated shortcuts for quick emails, GitHub redirects, and LinkedIn profile access.

## 🛠️ Built With

- **React 19** - Front-end library for building UI components.
- **TypeScript** - Type-safe programming language for scalable development.
- **Vite** - High-performance frontend bundler and build tool.
- **Tailwind CSS** - Utility-first styling framework for fluid layouts.
- **Motion (Framer Motion)** - Hardware-accelerated animations and transition effects.
- **Lucide React** - High-quality open-source UI icons.

## 💻 Getting Started

Follow these instructions to set up and run the portfolio website locally on your machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed (version 18.x or above is recommended).

### Installation

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/CoderKunal02/Portfolio.git
   cd Portfolio
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run in Development Mode:**
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:3000` to view the live application.

### Building for Production

To compile and optimize the application for deployment:

```bash
npm run build
```

This will generate a static production bundle in the `dist` directory which can be easily hosted on platforms like GitHub Pages, Vercel, Netlify, or AWS.
